Variant of a list
=================

A valid list only had valid data.
A invalid list had only invalid data or some valid data with invalid data.

```haskell
instance (Variant a) => Variant [a] where
  valid = do
    x <- valid
    xs <- valid
    (oneof . map return) [[], [x], x:xs]
  invalid = do
    x <- invalid
    xs <- invalid
    y <- valid
    ys <- valid
(oneof . map return) [[x], x:xs, x:ys, y:xs]
```
